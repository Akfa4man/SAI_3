# SAI_3
В проекте реализована нейронная сеть типа многослойного перцептрона (MLP), использующая алгоритм прямого распространения сигнала и обратного распространения ошибки (backpropagation). Для активации нейронов скрытого слоя применяется функция ReLU, а на выходном слое используется функция Softmax. Обучение сети выполняется методом мини-пакетного градиентного спуска (SGD) с использованием кросс-энтропийной функции потерь.
